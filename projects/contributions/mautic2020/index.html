Mautic - Google Season of Docs 2020

Proposal

My proposal for the Mautic project involved providing an experienced Technical Writer’s perspective to help amplify the quality and effectiveness of some of the existing tutorials, how-to-guides, best-practices, etc within the Mautic knowledgebase project.

Project scope: Season of Docs Ideas 2020 Mautic

  • Review the knowledgebase.
  • Update the tutorials and how-to-guides fit the Mautic 3x version.
  • Create resources for the newly created Mautic Community Knowledgebase.
  • Update the video tutorial to fit the Mautic 3x version.
  • Remove obsolete images and information.
  • Rewrite the knowledgebase to make it easier to understand.
  • Ensure proper implementation of the style guide/pattern of the knowledgebase documentation
  • Write a series of how-to tutorials and guides that will help Mautic users be more effective in setting up, configuring, and using the application.

Key learnings and deliverables

  • Reviewed the knowledge base and simplified the writing process.

SDK documentation

Context

Mindee offers client libraries which helps developers to quickly and seamlessly integrate their off-the-shelf APIs and custom APIs within their application.

Problem

Collaborate with the engineering team to edit and creating content for the Mindee client libraries. Also, assist in testing out the code for accuracy and consistency within the documentation.

Contribution

Over the course of the project, I

  • Collaborated with the team to review the existing SDK documentation for the different client libraries
  • Test the code samples for the different languages the client libraries are released to ensure accuracy and consistency in the docs
  • Create a document template for writing the client libraries to provide structure and flow to the SDK documentation

Impact

  • Helped customers understand how to make use of the client libraries by providing code samples to get them started.
  • Improved the navigation and information architecture of the SDK documentation.
